不知道 API Key 怎么获取?点这里看 1 分钟教程

① AI 模型 Key(审查用) — 默认用 DeepSeek:打开 platform.deepseek.com → 注册登录 → 左侧「API Keys」→ 点「创建 API key」→ 复制 sk- 开头的字符串,填到上方「AI 模型 API Key」。(需在 DeepSeek 账户里充值少量额度才能调用)

② 联网搜索 Key(找法规用) — 默认用 Tavily(注册即送免费额度):打开 app.tavily.com → 用邮箱或 Google 免费注册 → 登录后控制台首页就能看到 tvly- 开头的 API Key → 复制,填到「联网搜索 API Key」。

想换其他搜索服务?Serper(谷歌结果,有免费额度)或 Bing Web Search(在微软 Azure 门户申请)也行,去各自后台拿 Key 后,记得把上方「联网搜索服务」切到对应项。

你填的所有 Key 只存在你自己的浏览器里,本站不上传、不保存。
